Viewing a Virtual Interface
===========================

Scenarios
---------

After creating a virtual interface, you can view its basic information.

Procedure
---------

#. Log in to the management console.

#. Click |image1| in the upper left corner and select a region and a project.

#. Under **Network**, click **Direct Connect**.

#. In the navigation pane on the left, choose **Direct Connect** > **Virtual Interfaces**.

#. In the virtual interface list, click |image2| on the left of the target virtual interface to view its details.
